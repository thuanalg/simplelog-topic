3. APIs manual

Index Name Description Thread-safe
1 int spl_init_log( char *pathcfg)

  1. pathcfg: The path to configuring file below.

    ## ------------------------------------------------------------------------------
    #pathfoder: the folder to write log
    pathfoder=log_simple
    #level: 0-->> debug, 70: Info, 80: warning, 90: err, 100: Fatal
    level=0
    #buffsize(byte): size of memory for simple log, 1(MB)
    buffsize=1000000
    #rotation_size(byte): size of each of log files, 10(MB)
    rotation_size=10000000
    #end_configuring: Put in end of this file to read configuring file no more
    end_configuring=1
 Yes
2 spllog(level, fmt, ...) Level is:
  1. SPL_LOG_DEBUG
  2. SPL_LOG_INFO
  3. SPL_LOG_WARNING
  4. SPL_LOG_ERROR
  5. SPL_LOG_FATAL
Yes
3 int spl_finish_log() Finish log. Yes
